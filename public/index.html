URL Shortener Microservice

Allows the user to shorten URLS, check shortened URLS, and navigate to the destinations where they point.

Usage

You'll type one of these commands into the address bar, after the URL.

  • /shorten/[url] - Shortens the given URL.
  • /check/[id] - Checks the destination of the given shortened ID.
  • /[id] - Navigates to the URL mapped to the given ID.
